Chapitre 01 25 min Débutant

🚀 Introduction à JavaScript

Découvrir JavaScript, son histoire et comment l'utiliser

🔍 1.1 Qu'est-ce que JavaScript ?

JavaScript est un langage de programmation qui permet de rendre les pages web interactives. Il s'exécute directement dans le navigateur de l'utilisateur.

📖 Explication :

  • HTML = la structure (les os)
  • CSS = l'apparence (la peau)
  • JavaScript = l'interactivité (les muscles)
💡
À savoir : JavaScript n'a rien à voir avec Java. C'est une coïncidence marketing ! Il a été créé en 1995 par Brendan Eich en seulement 10 jours.

📥 1.2 Où placer JavaScript

1. Interne (dans la page)

<script>
    console.log("Bonjour !");
    alert("Bienvenue");
</script>

À placer avant la fermeture du </body>

2. Externe (recommandé)

<script src="script.js"></script>

Fichier .js séparé, plus propre et réutilisable

3. Inline (à éviter)

<button onclick="alert('Cliqué !')">
    Cliquez
</button>

❌ Mélange HTML/JS

🖥️ 1.3 La console (meilleur ami du développeur)

La console permet de voir les messages, tester du code et déboguer.

// Afficher des messages
console.log("Ceci est un message normal");
console.info("Information");
console.warn("Attention !");
console.error("Erreur !");

// Afficher des tableaux et objets
console.table(["pomme", "banane", "orange"]);
console.log({nom: "Jean", age: 25});

// Grouper des messages
console.group("Groupe");
console.log("Message 1");
console.log("Message 2");
console.groupEnd();

// Mesurer le temps d'exécution
console.time("Timer");
for (let i = 0; i < 1000000; i++) {}
console.timeEnd("Timer");
Astuce : Ouvrez la console avec F12 ou clic droit → "Inspecter" → onglet "Console". C'est là que vous passerez le plus clair de votre temps !

📝 1.4 Les commentaires

// Ceci est un commentaire sur une ligne

/*
   Ceci est un commentaire
   sur plusieurs lignes
   très utile pour expliquer
   des blocs de code complexes
*/

console.log("Bonjour"); // Commentaire en fin de ligne

🔚 1.5 Les points-virgules (;)

Les points-virgules sont optionnels en JavaScript, mais recommandés pour éviter des erreurs dans certains cas.

// Avec point-virgule (recommandé)
let a = 5;
let b = 10;
console.log(a + b);

// Sans point-virgule (fonctionne aussi)
let a = 5
let b = 10
console.log(a + b)

📦 Variables et types de données

Stocker et manipuler des données

📋 2.1 Déclarer des variables

Trois façons de déclarer des variables :

Mot-clé Portée Réassignable Hoisting
let Bloc {} ✅ Oui Non (Temporal Dead Zone)
const Bloc {} ❌ Non Non (Temporal Dead Zone)
var Fonction ✅ Oui ✅ Oui (mais à éviter) 
// Déclaration avec let (modifiable)
let age = 25;
age = 26; // OK

// Déclaration avec const (constante, non modifiable)
const nom = "Jean";
// nom = "Pierre"; // Erreur !

// var (ancienne façon, à éviter)
var ancienneVariable = "À ne plus utiliser";
💡
Règle : Utilisez const par défaut, et let seulement si vous devez réassigner la variable. N'utilisez jamais var dans du code moderne.

🔤 2.2 Les types de données primitifs

JavaScript a 7 types primitifs :

Type Description Exemple
number Nombres (entiers ou décimaux) let age = 25; let prix = 19.99;
string Chaînes de caractères let nom = "Jean"; let message = 'Bonjour';
boolean Vrai ou faux let estVrai = true; let estFaux = false;
undefined Valeur non définie let x; // undefined
null Valeur vide (volontairement) let y = null;
symbol Valeur unique let sym = Symbol('id');
bigint Grands entiers let grand = 12345678901234567890n; 
// Exemples concrets
let nom = "Jean";              // string
let age = 25;                  // number
let prix = 19.99;              // number (pas de différence entier/décimal)
let estMajeur = true;          // boolean
let resultat = undefined;      // undefined
let valeurVide = null;         // null

// Vérifier le type
console.log(typeof nom);        // "string"
console.log(typeof age);        // "number"
console.log(typeof estMajeur);  // "boolean"
console.log(typeof resultat);   // "undefined"
console.log(typeof valeurVide); // "object" (c'est un bug historique !)

🎯 Différence entre null et undefined :

  • undefined : la variable existe mais n'a pas de valeur
  • null : la variable existe et a volontairement une valeur vide

🔗 2.3 Travailler avec les chaînes

Plusieurs façons de combiner du texte :

let prenom = "Jean";
let nom = "Dupont";
let age = 25;

// Concaténation avec +
console.log("Je m'appelle " + prenom + " " + nom + " et j'ai " + age + " ans.");

// Template strings (avec backticks ``) - RECOMMANDÉ
console.log(`Je m'appelle ${prenom} ${nom} et j'ai ${age} ans.`);

// On peut mettre des expressions dans ${}
console.log(`L'année prochaine, j'aurai ${age + 1} ans.`);
Astuce : Toujours utiliser les template strings (avec backticks) pour les chaînes avec variables. C'est plus lisible et plus puissant.

⚖️ 3.2 Opérateurs de comparaison

Opérateur Description Exemple Résultat
==Égalité (valeur)5 == "5"true

!=Différence5 != 3true

⚠️
Règle d'or : Toujours utiliser === et !== pour éviter les conversions implicites surprenantes.

🎯 Pourquoi éviter ==

console.log(0 == false);     // true (surprenant !)
console.log("" == false);    // true (surprenant !)
console.log(null == undefined); // true (surprenant !)

console.log(0 === false);    // false (correct)
console.log("" === false);   // false (correct)
console.log(null === undefined); // false (correct)

🔗 3.3 Opérateurs logiques

Opérateur Description Exemple Résultat
&&ET (les deux doivent être vrais)true && truetrue
||OU (au moins un vrai)true || falsetrue
!NON (inverse)!truefalse
 
let age = 20;
let aPermis = true;

// Peut conduire ? (âge >= 18 ET a le permis)
let peutConduire = (age >= 18) && aPermis;
console.log(peutConduire); // true

// Est-ce un enfant ou un senior ? (âge < 12 OU age > 65)
let tarifReduit = (age < 12) || (age > 65);
console.log(tarifReduit); // false

// Est-il mineur ? (inverse de majeur)
let estMineur = !(age >= 18);
console.log(estMineur); // false

📝 3.4 Opérateurs d'assignation

let x = 10;

x += 5;   // équivalent à x = x + 5 (x = 15)
x -= 3;   // équivalent à x = x - 3 (x = 12)
x *= 2;   // équivalent à x = x * 2 (x = 24)
x /= 4;   // équivalent à x = x / 4 (x = 6)
x %= 4;   // équivalent à x = x % 4 (x = 2)

// Incrémentation et décrémentation
let compteur = 0;
compteur++; // compteur = compteur + 1 (1)
compteur--; // compteur = compteur - 1 (0)

🔄 Structures conditionnelles

Prendre des décisions dans le code

🔀 4.1 if, else if, else

let age = 18;
                                  if (age >= 18) {
    console.log("Vous êtes majeur");
} else {
    console.log("Vous êtes mineur");
}

// Avec plusieurs conditions
let note = 15;

if (note >= 16) {
    console.log("Excellent");
} else if (note >= 14) {
    console.log("Très bien");
} else if (note >= 12) {
    console.log("Bien");
} else if (note >= 10) {
    console.log("Passable");
} else {
    console.log("Insuffisant");
}

🎯 Exemple concret :

let heure = 14;

if (heure < 12) {
    console.log("Bonjour !");
} else if (heure < 18) {
    console.log("Bon après-midi !");
} else {
    console.log("Bonsoir !");
}

❓ 4.2 Opérateur ternaire (condition ? vrai : faux)

Raccourci pour les conditions simples.

// Avec if/else
let age = 20;
let statut;

if (age >= 18) {
    statut = "majeur";
} else {
    statut = "mineur";
}

// Avec ternaire (plus court)
let statut = (age >= 18) ? "majeur" : "mineur";

// On peut même l'utiliser directement dans un template
console.log(`Vous êtes ${age >= 18 ? "majeur" : "mineur"}`);
Attention : Le ternaire est pratique, mais à utiliser avec parcimonie. Si la logique est complexe, préférez if/else pour la lisibilité.

🔌 4.3 switch

Utile quand on a plusieurs valeurs possibles pour une même variable.

let jour = 3;
let nomJour;

switch (jour) {
    case 1:
        nomJour = "Lundi";
        break;
    case 2:
        nomJour = "Mardi";
        break;
    case 3:
        nomJour = "Mercredi";
        break;
    case 4:
        nomJour = "Jeudi";
        break;
    case 5:
        nomJour = "Vendredi";
        break;
    case 6:
        nomJour = "Samedi";
        break;
    case 7:
        nomJour = "Dimanche";
        break;
    default:
        nomJour = "Jour invalide";
}

console.log(nomJour); // Mercredi
⚠️
Ne pas oublier break ! Sans break, l'exécution continue sur le case suivant (effet de bord parfois voulu).

Exemple avec "fallthrough" (volontaire)

let note = "B";

switch (note) {
    case "A":
    case "B":
    case "C":
        console.log("Réussite");
        break;
    case "D":
    case "E":
        console.log("Échec");
        break;
    default:
        console.log("Note invalide");
}

🔄 Les boucles

Répéter des actions

🔢 5.1 La boucle for

Idéale quand on connaît le nombre d'itérations.

// Syntaxe : for (initialisation; condition; incrémentation)
for (let i = 0; i < 5; i++) {
    console.log(`Itération numéro ${i}`);
}

// Parcourir un tableau
let fruits = ["pomme", "banane", "orange"];
for (let i = 0; i < fruits.length; i++) {
    console.log(fruits[i]);
}

🎯 Exemples :

// Compter de 0 à 10 avec un pas de 2
for (let i = 0; i <= 10; i += 2) {
    console.log(i); // 0, 2, 4, 6, 8, 10
}

// Compter à l'envers
for (let i = 10; i >= 0; i--) {
    console.log(i); // 10, 9, 8...
}

🔄 5.2 La boucle while

S'exécute tant que la condition est vraie. À utiliser quand on ne connaît pas le nombre d'itérations à l'avance.

let compteur = 0;

while (compteur < 5) {
    console.log(`Compteur: ${compteur}`);
    compteur++;
}

// Attention aux boucles infinies !
// while (true) { // boucle infinie !
//     console.log("Attention !");
// }

🔁 5.3 La boucle do...while

S'exécute au moins une fois, puis continue tant que la condition est vraie.

let i = 0;

do {
    console.log(`i = ${i}`);
    i++;
} while (i < 5);

// Même avec une condition fausse au départ, ça s'exécute une fois
let j = 10;
do {
    console.log("Ce message s'affiche quand même");
    j++;
} while (j < 5);

⏹️ 5.4 break et continue

break (arrêter la boucle)

for (let i = 0; i < 10; i++) {
    if (i === 5) {
        break; // Sort de la boucle à i = 5
    }
    console.log(i); // 0,1,2,3,4
}

continue (sauter une itération)

for (let i = 0; i < 10; i++) {
    if (i % 2 === 0) {
        continue; // Saute les nombres pairs
    }
    console.log(i); // 1,3,5,7,9
}
