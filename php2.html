🏷️ Les tableaux associatifs

Stocker des données avec des clés nommées

➕ 7.1 Créer un tableau associatif

 "Dupont",
        "prenom" => "Jean",
        "age" => 25,
        "ville" => "Antananarivo"
    ];
    
    print_r($personne);
    // Array ( [nom] => Dupont [prenom] => Jean [age] => 25 [ville] => Antananarivo )
?>

On utilise => pour associer une clé à une valeur.

🔍 7.2 Accéder aux valeurs

 "Dupont",
        "prenom" => "Jean",
        "age" => 25
    ];
    
    echo $personne["nom"];     // "Dupont"
    echo $personne["prenom"];  // "Jean"
    echo $personne["age"];     // 25
    
    // Modifier
    $personne["age"] = 26;
    
    // Ajouter une nouvelle clé
    $personne["ville"] = "Antananarivo";
?>

🔄 7.3 Parcourir un tableau associatif

 "Dupont",
        "prenom" => "Jean",
        "age" => 25
    ];
    
    foreach ($personne as $cle => $valeur) {
        echo "$cle : $valeur
";
    }
    // nom : Dupont
    // prenom : Jean
    // age : 25
    
    // Pour les valeurs seulement
    foreach ($personne as $valeur) {
        echo "$valeur
";
    }
?>

⚙️ Fonctions de tableau

Manipuler les tableaux efficacement

➕ 8.1 Ajouter et supprimer des éléments

Fonction Description Exemple
array_push() Ajoute à la fin array_push($arr, "valeur")
array_pop() Supprime le dernier $dernier = array_pop($arr)
array_unshift() Ajoute au début array_unshift($arr, "valeur")
array_shift() Supprime le premier $premier = array_shift($arr)

🔍 8.2 Rechercher dans un tableau

Fonction Description Exemple
in_array() Vérifie si une valeur existe in_array("pomme", $fruits)
array_search() Cherche l'index d'une valeur $index = array_search("pomme", $fruits)
array_key_exists() Vérifie si une clé existe array_key_exists("nom", $personne) 
 "Dupont", "age" => 25];
    
    if (array_key_exists("nom", $personne)) {
        echo "La clé 'nom' existe";
    }
?>

📊 8.3 Trier les tableaux

Fonction Description
sort()Trie un tableau indexé par valeurs (croissant)
rsort()Trie un tableau indexé par valeurs (décroissant)
asort()Trie un tableau associatif par valeurs (croissant)
arsort()Trie un tableau associatif par valeurs (décroissant)
ksort()Trie un tableau associatif par clés (croissant)
krsort()Trie un tableau associatif par clés (décroissant)
 
 "Dupont",
        "prenom" => "Jean",

    ];

?>

🔧 8.4 Autres fonctions utiles

Fonction Description Exemple
count() Nombre d'éléments count($fruits)
implode() Tableau → chaîne implode(", ", $fruits)
explode() Chaîne → tableau explode(",", $chaine)
array_merge() Fusionne des tableaux array_merge($arr1, $arr2)
array_unique() Supprime les doublons array_unique($arr)
array_reverse() Inverse l'ordre array_reverse($arr)

🌐 Portée des variables (Scope)

Où les variables sont-elles accessibles ?

🌍 10.1 Globale vs Locale

⚠️
Important : Les variables définies dans une fonction ne sont accessibles qu'à l'intérieur de cette fonction.

🌐 10.2 Utiliser global

Pour accéder à une variable globale dans une fonction, on utilise global.

📦 10.3 Variables statiques (static)

Une variable static conserve sa valeur entre les appels à la fonction.

";
    }
    
    compteur();  // Appel numéro 1
    compteur();  // Appel numéro 2
    compteur();  // Appel numéro 3
    compteur();  // Appel numéro 4
    
    // Sans static
    function compteurNormal() {
        $cpt = 0;
        $cpt++;
        echo "Appel numéro $cpt
";
    }
    
    compteurNormal();  // Appel numéro 1
    compteurNormal();  // Appel numéro 1
    compteurNormal();  // Appel numéro 1
?>
