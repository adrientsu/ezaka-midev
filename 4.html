Chapitre 17 30 min Expert

✨ Pseudo-éléments

::before, ::after, ::first-letter, ::first-line

🔍 17.1 Qu'est-ce qu'un pseudo-élément ?

Les pseudo-éléments permettent de styliser des parties spécifiques d'un élément, sans avoir à ajouter de balises HTML supplémentaires. Ils commencent par :: (double deux-points).

💡
Différence :
  • Pseudo-classes (:) - État de l'élément (:hover, :focus)
  • Pseudo-éléments (::) - Partie de l'élément (::before, ::first-letter)

🎭 17.2 ::before et ::after (les plus utilisés)

Ces pseudo-éléments insèrent du contenu avant ou après le contenu de l'élément.

.element {
    position: relative;
}

.element::before {
    content: "🔥";           /* Obligatoire ! */
    margin-right: 5px;
    color: #FF7A45;
}

.element::after {
    content: "✨";
    margin-left: 5px;
}

La propriété content

content est obligatoire pour ::before et ::after. Peut prendre différentes valeurs :

  • content: "texte"; - Texte simple
  • content: ""; - Vide (utile pour les éléments décoratifs)
  • content: url('image.png'); - Image
  • content: attr(data-attr); - Valeur d'un attribut
  • content: counter(compteur); - Compteur
  • content: open-quote; - Guillemets

🎨 17.3 Utilisations créatives de ::before et ::after

Icônes décoratives

h1::before {
    content: "★";
    color: gold;
    margin-right: 10px;
}

Guillemets stylisés pour citations

blockquote::before {
    content: "«";
    font-size: 2em;
    color: #FF7A45;
    position: absolute;
    left: -20px;
    top: -10px;
}

blockquote::after {
    content: "»";
    font-size: 2em;
    color: #FF7A45;
    position: absolute;
    right: -20px;
    bottom: -10px;
}

Bordures décoratives

.titre-section {
    position: relative;
    text-align: center;
}

.titre-section::before {
    content: "";
    position: absolute;
    bottom: -10px;
    left: 50%;
    transform: translateX(-50%);
    width: 50px;
    height: 3px;
    background: #FF7A45;
}

Tooltips (infobulles) avec CSS pur

.tooltip {
    position: relative;
    cursor: help;
}

.tooltip:hover::after {
    content: attr(data-tooltip);
    position: absolute;
    bottom: 100%;
    left: 50%;
    transform: translateX(-50%);
    background: #1F2D3D;
    color: white;
    padding: 5px 10px;
    border-radius: 4px;
    white-space: nowrap;
    font-size: 0.8rem;
}

📝 17.4 ::first-letter et ::first-line

Permettent de styliser la première lettre ou la première ligne d'un texte.

p::first-letter {
    font-size: 3em;
    color: #FF7A45;
    float: left;
    margin-right: 5px;
    font-weight: bold;
}

p::first-line {
    font-weight: bold;
    color: #1F2D3D;
}

Ceci est un paragraphe avec une première lettre stylisée. C'est un effet typographique classique comme dans les livres anciens.

Ceci est un paragraphe où seule la première ligne est en gras. Si le texte est long, la première ligne seule sera modifiée jusqu'au premier retour à la ligne automatique.

🖱️ 17.5 ::selection et ::placeholder

Stylise la sélection de texte et les placeholders de formulaires.

/* Style du texte sélectionné */
::selection {
    background: #FF7A45;
    color: white;
}

/* Style du placeholder */
input::placeholder {
    color: #6B7280;
    font-style: italic;
    opacity: 0.8;
}

Sélectionnez ce texte pour voir le style personnalisé !

🎯 Pseudo-classes avancées

:nth-child, :not, :is, :where, :has

📌 18.1 Rappel des pseudo-classes de base

Pseudo-classes déjà vues : :hover, :focus, :active, :first-child, :last-child, etc.

🔢 18.2 :nth-child() et :nth-of-type()

Sélectionne des éléments selon leur position parmi leurs frères.

Syntaxe Description
:nth-child(3)Troisième enfant
:nth-child(odd)Enfants impairs (1, 3, 5...)
:nth-child(even)Enfants pairs (2, 4, 6...)
:nth-child(2n+1)Impairs (équivalent à odd)
:nth-child(3n)Tous les 3 éléments (3, 6, 9...)
:nth-child(n+4)À partir du 4ème élément
:nth-child(-n+3)Les 3 premiers éléments
:nth-of-type(2)Deuxième élément du même type
 
/* Exemples */
li:nth-child(odd) {
    background: #F6EDE7;
}

li:nth-child(3n+1) {
    color: #FF7A45;
}

p:nth-of-type(2) {
    font-weight: bold;  /* Deuxième paragraphe (ignorer les autres types) */
}

🎯 Exemple visuel :

  • Élément 1 (impair)
  • Élément 2 (pair)
  • Élément 3 (spécial)
  • Élément 4 (pair)
  • Élément 5 (impair)

🚫 18.3 :not() - la négation

Sélectionne les éléments qui ne correspondent pas au sélecteur donné.

/* Tous les paragraphes sauf ceux avec la classe .special */
p:not(.special) {
    color: #6B7280;
}

/* Tous les liens sauf ceux dans le footer */
a:not(footer a) {
    color: #FF7A45;
}

/* Tous les inputs sauf les boutons */
input:not([type="submit"]) {
    border: 1px solid #6B7280;
}

🔗 18.4 :is() et :where() (regroupement)

Permettent de regrouper plusieurs sélecteurs pour simplifier le code.

Sans :is()

header h1, header h2, header h3 {
    font-family: 'Arial', sans-serif;
}

Avec :is()

header :is(h1, h2, h3) {
    font-family: 'Arial', sans-serif;
}

Différence entre :is() et :where() : :where() a une spécificité de 0 (toujours), tandis que :is() prend la spécificité la plus élevée de ses arguments.

/* Plus d'exemples */
/* Cibler les titres dans header ET footer */
:is(header, footer) :is(h1, h2, h3) {
    color: #1F2D3D;
}

/* Formulaires */
:is(input, textarea, select):focus {
    outline: 2px solid #FF7A45;
}

👪 18.5 :has() - le sélecteur parent

:has() permet de sélectionner un élément qui contient un certain élément enfant. C'est une nouveauté puissante.

/* Sélectionner les articles qui contiennent une image */
article:has(img) {
    border: 2px solid #FF7A45;
}

/* Sélectionner les liens qui contiennent une icône */
a:has(i) {
    padding-left: 5px;
}

/* Sélectionner les formulaires qui contiennent une erreur */
form:has(.error) {
    border-color: red;
}

/* Style différent pour les éléments qui suivent un élément */
h2:has(+ p) {
    margin-bottom: 0;  /* Si le h2 est suivi d'un p */
}

Chapitre 19 30 min Expert

🧮 Fonctions CSS avancées

calc(), clamp(), min(), max()

🧮 19.1 calc() - Calculs dynamiques

calc() permet d'effectuer des opérations mathématiques avec différentes unités.

.element {
    width: calc(100% - 40px);
    height: calc(100vh - 80px);
    margin: calc(1rem + 5px);
    font-size: calc(16px + 0.5vw);
    top: calc(50% - 50px);
}

🎯 Exemples concrets :

  • Sidebar fixe + contenu fluide : width: calc(100% - 250px);
  • Centrage parfait : top: calc(50% - 50px); left: calc(50% - 50px);
  • Typographie fluide : font-size: calc(14px + 0.5vw);
  • Colonnes avec gouttières : width: calc(33.33% - 10px);
Opérateurs : + - * / . Attention aux espaces autour des + et - !

📏 19.2 min() et max() - Choisir la plus petite/grande valeur

Sélectionnent la valeur minimale ou maximale parmi une liste.

/* min() - ne dépasse jamais une certaine taille */
.conteneur {
    width: min(90%, 1200px);
    /* = prend 90% de la largeur, mais jamais plus de 1200px */
}

.image {
    width: min(100%, 300px);
    /* = s'adapte mais ne dépasse pas 300px */
}

/* max() - a toujours au moins une certaine taille */
.titre {
    font-size: max(24px, 5vw);
    /* = au moins 24px, mais peut grandir avec 5vw */
}

.sidebar {
    width: max(200px, 20%);
    /* = au moins 200px, mais peut être plus large si 20% est plus grand */
}

🎯 min() pour le responsive :

Cette boîte fait 90% de la largeur, max 400px

📐 19.3 clamp() - La fonction ultime

clamp(min, préféré, max) définit une valeur entre un minimum et un maximum.

h1 {
    /* font-size: clamp(minimum, préféré, maximum) */
    font-size: clamp(24px, 5vw, 48px);
    /* = entre 24px et 48px, de préférence 5vw */
}

.conteneur {
    width: clamp(300px, 80%, 1200px);
    /* = entre 300px et 1200px, de préférence 80% */
}

.padding {
    padding: clamp(10px, 3vw, 30px);
}

🎯 Typographie responsive parfaite :

Ce texte s'adapte : minimum 16px, maximum 24px, taille préférée 3vw. Redimensionnez la fenêtre !

🔗 19.4 Combiner les fonctions

On peut combiner ces fonctions pour des layouts complexes.

.layout {
    /* Grille responsive avec clamp */
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(clamp(200px, 25%, 300px), 1fr));
}

.espacement {
    /* Marge qui s'adapte mais jamais trop petite/grande */
    margin: clamp(10px, 5vh, 50px) clamp(10px, 5vw, 50px);
}

.boite {
    /* calc + clamp */
    width: calc(clamp(300px, 50%, 600px) - 20px);
}
