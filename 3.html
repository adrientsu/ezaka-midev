Chapitre 11 30 min Avancé

🔄 Les transitions CSS

Animer les changements d'état en douceur

🎬 11.1 Qu'est-ce qu'une transition ?

Les transitions CSS permettent de changer progressivement la valeur d'une propriété CSS sur une durée donnée. Au lieu de passer instantanément d'un état à un autre, on obtient un mouvement fluide.

💡
Principe : Une transition est un changement d'état (souvent au survol, au focus, ou lors de l'ajout/suppression d'une classe) qui se fait progressivement.

📝 11.2 Syntaxe de base

.element {
    transition-property: all;        /* Propriété à animer (ou all) */
    transition-duration: 0.3s;      /* Durée de l'animation */
    transition-timing-function: ease; /* Courbe d'accélération */
    transition-delay: 0s;            /* Délai avant démarrage */
    
    /* Raccourci */
    transition: all 0.3s ease 0s;
    transition: background-color 0.3s ease;
}

🎯 Exemple simple :

Survolez

La boîte change de couleur et s'agrandit en douceur au survol.

🎯 11.3 transition-property

Définit quelle(s) propriété(s) seront animées.

/* Une seule propriété */
.bouton {
    transition-property: background-color;
    transition-duration: 0.3s;
}

/* Plusieurs propriétés (séparées par des virgules) */
.bouton {
    transition-property: background-color, transform, box-shadow;
    transition-duration: 0.3s, 0.5s, 0.2s;
}

/* Toutes les propriétés */
.bouton {
    transition: all 0.3s ease;
}
Astuce : N'animez que les propriétés qui peuvent être animées (couleurs, dimensions, positions, transformations). Évitez d'animer `width`, `height`, `margin`, `padding` qui peuvent causer des repositionnements coûteux.

⏱️ 11.4 transition-duration

Définit la durée de l'animation, en secondes (s) ou millisecondes (ms).

.rapide {
    transition: all 0.1s ease;  /* Très rapide */
}

.normal {
    transition: all 0.3s ease;  /* Vitesse standard */
}

.lent {
    transition: all 1s ease;    /* Lent */
}

}

⏰ 11.6 transition-delay

Définit un délai avant le début de l'animation.

.avec-delai {
    transition: all 0.5s ease 0.2s;  /* Attend 0.2s avant de commencer */
}

🎭 11.7 Transitions multiples

On peut avoir des transitions différentes pour différentes propriétés.

.bouton {
    background-color: #FF7A45;
    transform: scale(1);
    
    transition-property: background-color, transform;
    transition-duration: 0.3s, 0.5s;
    transition-timing-function: ease, ease-out;
    transition-delay: 0s, 0.1s;
    
    /* Raccourci */
    transition: background-color 0.3s ease 0s, transform 0.5s ease-out 0.1s;
}

.bouton:hover {
    background-color: #1F2D3D;
    transform: scale(1.1);
}

📋 11.8 Quelles propriétés peut-on animer ?

Toutes les propriétés qui ont des valeurs intermédiaires possibles :

✅ Animables :

  • Couleurs (color, background-color, border-color)
  • Dimensions (width, height, font-size)
  • Positions (top, right, bottom, left)
  • Transformations (transform, rotate, scale)
  • Opacité (opacity)
  • Filtres (filter)
  • Box-shadow, text-shadow
  • Flexbox (flex-grow, flex-shrink)
  • Grid (grid-template-columns, grid-gap)

❌ Non animables :

  • display (pas de valeurs intermédiaires)
  • font-family (pas de transition)
  • visibility (peut être animé avec une astuce)
  • overflow
  • position
  • z-index

🔄 Transformations 2D

rotate, scale, translate, skew

🔧 12.1 Qu'est-ce que transform ?

La propriété transform permet de modifier la position, la taille, la rotation et l'inclinaison d'un élément, sans affecter la disposition des autres éléments.

💡
Important : Les transformations sont appliquées après la mise en page. L'élément conserve son espace d'origine dans le flux (contrairement à position qui peut le sortir du flux).

📏 12.2 translate (déplacement)

Déplace un élément de sa position d'origine.

.element {
    /* Déplacement horizontal */
    transform: translateX(50px);
    
    /* Déplacement vertical */
    transform: translateY(30px);
    
    /* Déplacement horizontal et vertical */
    transform: translate(50px, 30px);
    
    /* Pourcentages (relatifs à la taille de l'élément) */
    transform: translate(50%, 20%);
}

🎯 Exemple :

Original
Translate
Astuce : translate(50%, 50%) permet de décaler un élément de la moitié de sa propre taille. Très utile pour centrer avec position absolute !

🔄 12.3 rotate (rotation)

Fait pivoter un élément autour de son centre (par défaut).

.element {
    /* Rotation en degrés */
    transform: rotate(45deg);   /* Sens horaire */
    transform: rotate(-30deg);  /* Sens anti-horaire */
    
    /* Autres unités possibles */
    transform: rotate(0.5turn);  /* 0.5 tour = 180deg */
    transform: rotate(3.1416rad); /* Radians */
}

🎯 Exemples :

15°
45°
90°
-20°

📊 12.4 scale (mise à l'échelle)

Agrandit ou rétrécit un élément.

.element {
    /* Facteur d'échelle uniforme */
    transform: scale(1.5);     /* Agrandit de 50% */
    transform: scale(0.5);     /* Réduit de moitié */
    
    /* Échelle différente horizontalement et verticalement */
    transform: scale(1.5, 0.8);  /* Largeur x1.5, hauteur x0.8 */
    
    /* Fonctions séparées */
    transform: scaleX(2);      /* Étire horizontalement */
    transform: scaleY(1.5);     /* Étire verticalement */
}

🎯 Exemples :

scale(1)
scale(1.2)
scale(0.8)
scale(1.5,0.8)

📐 12.5 skew (inclinaison)

Incline un élément sur l'axe horizontal ou vertical.

.element {
    /* Inclinaison horizontale */
    transform: skewX(20deg);
    
    /* Inclinaison verticale */
    transform: skewY(15deg);
    
    /* Inclinaison horizontale et verticale */
    transform: skew(20deg, 10deg);
}

🎯 Exemples :

Normal
skewX(15°)
skewY(15°)
skew(15°,10°)

🔗 12.6 Combiner plusieurs transformations

On peut appliquer plusieurs transformations à la fois, dans l'ordre donné.

.element {
    /* L'ordre compte ! */
    transform: translate(50px, 30px) rotate(45deg) scale(1.2);
}

transform-origin (origine de la transformation)

Change le point d'origine des transformations (par défaut, le centre).

.element {
    /* Mots-clés */
    transform-origin: top left;
    transform-origin: center;    /* Par défaut */
    transform-origin: right bottom;
    
    /* Coordonnées */
    transform-origin: 50% 50%;    /* Centre */
    transform-origin: 0 0;          /* Coin supérieur gauche */
    transform-origin: 100% 100%;    /* Coin inférieur droit */
}

🎯 Différence d'origine :

Centre
Top left

🎲 Transformations 3D

Perspective, rotations 3D, translate3d

🔮 13.1 La troisième dimension

Les transformations 3D ajoutent un axe Z (profondeur) aux axes X (horizontal) et Y (vertical) des transformations 2D.

💡
Principe : Pour créer un effet 3D réaliste, il faut :
  • Définir une perspective (point de fuite)
  • Appliquer des transformations 3D (rotateX, rotateY, translateZ)
  • Parfois, activer le rendu 3D des faces (transform-style)

👁️ 13.2 La perspective

La perspective définit la distance entre le spectateur et le plan de l'écran. Une petite valeur donne un effet plus prononcé.

/* Sur le conteneur parent */
.scene {
    perspective: 800px;    /* Distance du spectateur */
    perspective-origin: center;  /* Point de vue */
}

.cube {
    transform: rotateY(45deg);
}

/* Ou directement sur l'élément (moins réaliste) */
.element {
    transform: perspective(800px) rotateY(45deg);
}

🎯 Effet de perspective :

Sans perspective :

rotateY(45°)

Avec perspective (400px) :

rotateY(45°)

Avec perspective (200px) :

rotateY(45°)

🔄 13.3 Rotations 3D

On peut pivoter autour des axes X, Y et Z.

.element {
    /* Rotation autour de X (horizontal) */
    transform: rotateX(45deg);  /* Bascule vers l'avant/arrière */
    
    /* Rotation autour de Y (vertical) */
    transform: rotateY(45deg);  /* Tourne comme une porte */
    
    /* Rotation autour de Z (identique à rotate 2D) */
    transform: rotateZ(45deg);
    
    /* Combinaison */
    transform: rotateX(30deg) rotateY(45deg);
    
    /* Raccourci 3D */
    transform: rotate3d(1, 1, 0, 45deg);  /* Vecteur (x,y,z) + angle */
}

🎯 Exemples de rotations 3D :

rotateX
rotateY
rotateX+Y

📏 13.4 translate3d et translateZ

Déplacement sur l'axe Z (profondeur).

.element {
    /* Déplacement sur Z (rapproche ou éloigne) */
    transform: translateZ(50px);   /* Rapproche */
    transform: translateZ(-50px);  /* Éloigne */
    
    /* Déplacement sur X, Y, et Z */
    transform: translate3d(20px, 30px, 50px);
    
    /* Combinaison avec perspective */
    transform: perspective(400px) translateZ(100px);
}

🎯 Effet de profondeur :

Z:0
Z:50px
Z:100px

🧊 13.5 transform-style (préserver la 3D)

Permet aux enfants d'un élément 3D de conserver leur position dans l'espace 3D.

.parent {
    transform-style: preserve-3d;   /* Les enfants sont en 3D */
    transform-style: flat;            /* Par défaut, aplatit les enfants */
}

preserve-3d :

Contrôle si la face arrière d'un élément est visible quand il est tourné.

.carte {
    backface-visibility: hidden;   /* Cache la face arrière */
    backface-visibility: visible;  /* Par défaut */
}

🎬 Animations CSS

@keyframes, animation-name, duration, iteration

🎞️ 14.1 Différence entre transition et animation

Transitions

  • Nécessitent un changement d'état (hover, focus)
  • Un seul aller-retour possible
  • Simples à mettre en place
  • Idéales pour les micro-interactions

Animations

  • Peuvent démarrer automatiquement
  • Plusieurs étapes (keyframes)
  • Répétitions possibles
  • Contrôle plus fin
  • Idéales pour les effets visuels complexes

🎯 14.2 @keyframes (étapes de l'animation)

Définit les étapes de l'animation avec des pourcentages ou des mots-clés.

@keyframes nom-animation {
    /* Étapes avec pourcentages */
    0% {
        opacity: 0;
        transform: translateX(-100px);
    }
    50% {
        opacity: 0.5;
        transform: translateX(50px);
    }
    100% {
        opacity: 1;
        transform: translateX(0);
    }
}

/* Mots-clés from et to (équivalent à 0% et 100%) */
@keyframes apparition {
    from {
        opacity: 0;
    }
    to {
        opacity: 1;
    }
}

📝 14.3 animation-name et animation-duration

Applique l'animation à un élément.

.element {
    animation-name: apparition;      /* Nom de l'animation */
    animation-duration: 2s;          /* Durée */
}

🎯 Animation simple :

Animation

🔄 14.4 animation-iteration-count (répétitions)

Nombre de répétitions de l'animation.

.element {
    animation-iteration-count: 1;        /* Une seule fois (défaut) */
    animation-iteration-count: 3;        /* 3 fois */
    animation-iteration-count: infinite;   /* Infini */
}
Infini

↔️ 14.5 animation-direction

Direction de l'animation.

Valeur Description
normal0% → 100% (défaut)
reverse100% → 0% (inverse)
alternate0% → 100% puis 100% → 0% (aller-retour)
alternate-reverseInverse puis normal
 
.element {
    animation: mouvement 2s infinite alternate;
}

📈 14.6 animation-timing-function

Mêmes valeurs que pour les transitions (ease, linear, etc.).

.element {
    animation: mouvement 2s ease-in-out infinite;
}

⏰ 14.7 animation-delay

Délai avant le début de l'animation (peut être négatif !).

.element {
    animation-delay: 1s;    /* Commence après 1s */
    animation-delay: -1s;   /* Commence à 1s dans l'animation */
}

⏯️ 14.8 animation-fill-mode

Définit le style de l'élément avant/après l'animation.

Valeur Description
noneRetour à l'état initial après l'animation (défaut)
forwardsGarde le style de la dernière keyframe
backwardsApplique le style de la première keyframe pendant le délai
bothCombinaison de forwards et backwards
 
.element {
    animation: apparition 2s forwards;  /* Reste visible */
}

⏸️ 14.9 animation-play-state

Contrôle si l'animation est en cours ou en pause.

.element {
    animation: rotation 3s infinite;
}

.element:hover {
    animation-play-state: paused;  /* Met en pause au survol */
}

📝 14.10 La propriété raccourcie animation

.element {
    /* nom durée timing delay iteration direction fill-mode */
    animation: monAnimation 2s ease 1s infinite alternate forwards;
    
    /* Plusieurs animations */
    animation: anim1 2s ease, anim2 3s linear infinite;
}

✨ Filtres et effets

blur, brightness, contrast, drop-shadow

🔍 15.1 La propriété filter

La propriété filter permet d'appliquer des effets graphiques (flou, luminosité, contraste, etc.) à un élément, comme dans les logiciels de retouche d'image.

🎨 15.2 Les types de filtres

Filtre Description Exemple
blur(px) Flou gaussien filter: blur(5px);
brightness(%) Luminosité (0=noir, 1=normal, >1=plus lumineux) filter: brightness(0.5);
contrast(%) Contraste (0=gris, 1=normal, >1=contraste élevé) filter: contrast(2);
grayscale(%) Niveaux de gris (0=couleur, 1=noir et blanc) filter: grayscale(1);
hue-rotate(deg) Rotation de la teinte (cercle chromatique) filter: hue-rotate(90deg);
invert(%) Inversion des couleurs filter: invert(1);
opacity(%) Transparence (similaire à opacity mais peut être animé) filter: opacity(0.5);
saturate(%) Saturation (0=non saturé, 1=normal, >1=sursaturé) filter: saturate(2);
sepia(%) Effet sépia (vieux photos) filter: sepia(1);
drop-shadow() Ombre portée (différente de box-shadow) filter: drop-shadow(5px 5px 5px rgba(0,0,0,0.5));

    filter: brightness(0.8) contrast(1.2) grayscale(0.5) blur(1px);

backdrop-filter applique un filtre à la zone derrière l'élément (effet de verre dépoli).

Chapitre 16 25 min Avancé

📦 Variables CSS (Custom Properties)

Déclarer et utiliser des variables pour un code plus maintenable

🔧 16.1 Pourquoi utiliser des variables CSS ?

Les variables CSS (ou custom properties) permettent de stocker des valeurs réutilisables dans toute la feuille de style, facilitant la maintenance et les changements globaux.

💡
Avantages :
  • Un seul endroit à modifier pour changer une couleur/thème
  • Plus facile à maintenir
  • Peuvent être modifiées par JavaScript
  • Héritage et portée (comme en programmation)

📝 16.2 Syntaxe de base

/* Déclaration (généralement dans :root) */
:root {
    --couleur-principale: #FF7A45;
    --couleur-secondaire: #1F2D3D;
    --espacement: 20px;
    --police-titre: 'Arial', sans-serif;
}

/* Utilisation */
.bouton {
    background-color: var(--couleur-principale);
    color: white;
    padding: var(--espacement);
}

h1 {
    font-family: var(--police-titre);
    color: var(--couleur-secondaire);
}

🎯 16.3 Portée et héritage

Les variables sont héritées et peuvent être redéfinies dans des contextes plus spécifiques.

:root {
    --couleur-texte: #1F2D3D;  /* Globale */
}

.carte {
    --couleur-texte: #FF7A45;  /* Redéfinition locale */
    color: var(--couleur-texte);  /* Orange dans la carte */
}

p {
    color: var(--couleur-texte);  /* Bleu (hérité de :root) */
}

🛡️ 16.4 Valeurs par défaut

On peut spécifier une valeur par défaut si la variable n'est pas définie.

.element {
    /* Si --couleur-inexistante n'existe pas, utilise #FF7A45 */
    color: var(--couleur-inexistante, #FF7A45);
    
    /* Avec une variable existante, utilise sa valeur */
    background: var(--couleur-principale, #1F2D3D);
}

⚙️ 16.5 Modifier les variables avec JavaScript

Les variables CSS peuvent être modifiées dynamiquement, permettant de créer des thèmes, etc.

// JavaScript
// Changer une variable globale
document.documentElement.style.setProperty('--couleur-principale', '#FF9933');

// Changer une variable sur un élément spécifique
document.querySelector('.carte').style.setProperty('--couleur-texte', 'blue');

// Lire la valeur d'une variable
const couleur = getComputedStyle(document.documentElement)
    .getPropertyValue('--couleur-principale').trim();

🌓 16.6 Exemple : Thème clair/sombre

:root {
    --bg-color: #F6EDE7;
    --text-color: #1F2D3D;
    --primary-color: #FF7A45;
}

body.dark-theme {
    --bg-color: #1F2D3D;
    --text-color: #F6EDE7;
    --primary-color: #FF9933;
}

body {
    background-color: var(--bg-color);
    color: var(--text-color);
}

.bouton {
    background: var(--primary-color);
}

✍️ 16.7 Exercices pratiques

Exercice 1 : Système de couleurs

Objectif : Créer un système de couleurs avec variables.

Instructions :

  1. Déclarez les couleurs : primaire, secondaire, texte, fond
  2. Créez un bouton, une carte, un titre utilisant ces variables
  3. Changez les couleurs à un seul endroit

Exercice 2 : Thème interchangeable

Objectif : Créer deux thèmes (clair/sombre) avec variables.

Instructions :

  1. Définissez un thème clair et un thème sombre
  2. Ajoutez un bouton qui bascule la classe sur body
  3. Utilisez uniquement des variables pour les couleurs

🏆 Projet récapitulatif (Leçons 11-16)

Créez une landing page animée

Utilisez toutes les notions apprises dans les leçons 11 à 16.

Structure demandée :

  • Header avec navigation sticky
  • Section hero avec image de fond et titre animé (apparition)
  • Section features avec 3 cartes qui s'animent au scroll (transitions)
  • Section portfolio avec galerie d'images (filtres au survol)
  • Section témoignages avec des cartes qui tournent en 3D
  • Footer avec liens

Contraintes techniques :

  • Utilisez des variables CSS pour les couleurs
  • Ajoutez des transitions sur tous les boutons et cartes
  • Créez une animation @keyframes pour le titre principal
  • Utilisez des filtres sur les images (grayscale qui disparaît au survol)
  • Ajoutez un effet 3D sur les témoignages (rotateY léger au survol)
  • Implémentez un thème clair/sombre avec JavaScript (optionnel)
