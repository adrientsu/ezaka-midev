🚀 Introduction à PHP

Découvrir PHP, son histoire et comment l'utiliser

🔍 1.1 Qu'est-ce que PHP ?

PHP (Hypertext Preprocessor) est un langage de programmation côté serveur, spécialement conçu pour le développement web. Il permet de générer des pages dynamiques et d'interagir avec des bases de données.

📖 Explication :

  • Côté client (navigateur) : HTML, CSS, JavaScript
  • Côté serveur : PHP, Python, Java, etc.

PHP s'exécute sur le serveur avant d'envoyer le résultat au navigateur. Le code PHP n'est pas visible dans la source de la page.

💡
À savoir : PHP a été créé en 1994 par Rasmus Lerdorf. Il équipe aujourd'hui plus de 75% des sites web (Facebook, Wikipedia, WordPress).

📦 Variables et types de données

Stocker et manipuler des données en PHP

📋 2.1 Déclaration des variables

📌
Règles pour les noms de variables :
  • Commence toujours par $
  • Suivi d'une lettre ou underscore (_)
  • Pas de chiffre au début
  • Pas d'espaces, pas de caractères spéciaux
  • Sensible à la casse ($Nom ≠ $nom)
